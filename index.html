Ensure you have python3 installed: sudo apt install python3
Ensure you have pip installed: sudo apt install python3-pip

Due to the Guelph linux servers being externally managed, a virtual enviornment on the sever is created and worked in, in order to install and use pytest. To install a virtual environment type the following into the correct directory: python3 -m venv myenv

In order to activate the virtual environment, in order to allow you to work within a controlled environment where installed packages won't affect the system-wide Python installation, type: source myenv/bin/activate

In order to install the Pytest framework inside the activated virtual environment, type: pip install pytest

Once you are done developing or testing within the virtual environment, its good practice to deactivate the environment. Although this is optional, its highly recommended. To do so, type: deactivate
